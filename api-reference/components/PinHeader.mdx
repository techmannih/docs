A PinHeader component to define pin count and spacing for headers.

## Example Usage

```tsx
export const ExamplePinHeaderUsage = () => (
    <pinheader
        name="P1"
        pinCount={4}
        footprint="pinrow4"
        schRotation={90}
        facingDirection="left"
      />
)
```
